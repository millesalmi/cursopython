CÁLCULO DE MÉDIA DE TRÊS NÚMEROS

Nesse programa, nós iremos calcular a média entre três números. É um programa simples, e pode ser realizado rapidamente para praticar.

# Solicita ao usuário que insira três números

num1 = float(input("Digite o primeiro número: "))

num2 = float(input("Digite o segundo número: "))

num3 = float(input("Digite o terceiro número: "))

# Calcula a média

media = (num1 + num2 + num3) / 3

# Exibe o resultado

print(f"A média dos três números é: {media}")
