INTRODUÇÃO À LINGUAGEM PYTHON

Python é uma linguagem de programação de alto nível, ou seja, com sintaxe mais simplificada e próxima da linguagem humana, utilizada nas mais diversas aplicações, como desktop, web, servidores e ciência de dados. Saiba mais sobre ela, sua origem e principais vantagens neste guia para iniciantes.

O Python foi lançado no início da década de 90 pelo programador e matemático holandês Guido Van Rossum. A linguagem foi projetada para dar ênfase no trabalho do desenvolvedor, facilitando a escrita de um código limpo, simples e legível, tanto em aplicações menores quanto em programas mais complexos. A linguagem oferece recursos como tipagem dinâmica e forte (tipo de dado do valor deve ser do mesmo tipo da variável), orientação a objetos, multiparadigmas (programação funcional e imperativa), além de recursos poderosos em biblioteca padrão e via módulos e frameworks desenvolvidos pela comunidade. Seu código é aberto e a utilização é gratuita, rodando em praticamente qualquer sistema operacional.

CONCEITOS BÁSICOS

SINTAXE

Python foi originalmente desenvolvido como uma linguagem de ensino, mas sua facilidade de uso e sintaxe limpa levaram-no a ser adotado por iniciantes e especialistas. A sintaxe da linguagem de programação Python é o conjunto de regras que define como um programa Python será escrito e interpretado (tanto pelo sistema de execução quanto por leitores humanos).

ESTRUTURA DO PROGRAMA

Um programa em python é construído a partir de blocos de código. Um bloco é um pedaço do texto do programa Python que é executado como uma unidade. A seguir estão os blocos: módulo, corpo de função e definição de classe. Cada comando digitado interativamente é um bloco. Um arquivo de script é um bloco de código.Um bloco de código é executado em um quadro de execução, este contém algumas informações administrativas e determina onde e como a execução irá continuar após a conclusão do bloco de código.

VARIÁVEIS

As variáveis na linguagem Python são elementos fundamentais para o armazenamento e manipulação de dados dentro de um programa. Elas funcionam como “caixas” que guardam valores, que podem ser acessados e modificados durante o programa. Para criar uma variável, é necessário atribuir um valor a ela. Por exemplo, para criar uma variável mês e atribuir o valor “09” basta:

mes = 09

Existem algumas regras para criar uma variável:

  • devem iniciar com uma letra ou caractere sublinhado (_);

  • pode conter letras, números e o caractere (_);

  • palavras como “if”, “for” e “while” não podem ser utilizadas para variáveis;

TIPOS DE VARIÁVEIS

Na linguagem Python, podemos ter diversos tipos de variáveis para diferentes usos dentro da programação. Entre elas temos os seguintes tipos:

  • Inteiros (int): são variáveis para números inteiros, sem a parte decimal. Podem ser negativos ou positivos, como: 1; 67; -78.

  • Número de ponto flutuante (float): são usadas para números com a parte decimal. Como: 3.14; -1.5.

  • Números complexos (complex): são utilizadas para representar números com uma parte real e uma parte imaginária. São definidos na forma a + bi, onde a e b são números reais, e i é a unidade imaginária. Por exemplo: 2 + 3j; -1 + 2j.

  • Strings (str): são utilizadas para representar sequências de caracteres. São definidas entre aspas simples ou duas.Por exemplo: “Olá”.

  • Listas (list): são usadas para armazenar uma coleção ordenada de elementos, podem conter diferentes tipos de dados e podem ser modificadas. Exemplo: [1,2,3].

  • Teplas (tuple): são bem parecidas com as listas, mas não podem ser modificadas após a sua criação. Exemplo: (1,2,3).

  • Dicionários (dict): são utilizados para armazenar dados em pares chave-valor. Cada elemento em um dicionário é acessado por uma única chave. Por exemplo: (“nome”: “João”, “idade”: “30”).

  • Booleanos (bool): representam os valores lógicos verdadeiro (true) e falso (false) . Elas são frequentemente usadas em expressões condicionais e loops para controlar o fluxo do programa.

  • Conjuntos (set): em Python, são usados para armazenar coleções não ordenadas de elementos únicos. Eles são úteis quando você precisa verificar se um elemento está presente em uma coleção ou quando deseja eliminar duplicatas de uma lista.

As variáveis podem ter diversas aplicações, e podem ser utilizadas de maneiras diferentes. A seguir temos alguns exemplos de utilização de variáveis.

SOMA DE DOIS NÚMEROS

a = 10

b = 5

soma = a + b

print (soma)

Nesse exemplo, criamos as variáveis “a” e “b” e atribuímos a elas os valores 10 e 5, respectivamente. Em seguida, criamos uma terceira variável chamada “soma” e atribuímos a ela o resultado da soma entre “a” e “b”. Por fim, utilizamos a função “print()” para exibir o valor da variável “soma” na saída do programa.

ÁREA DE UM TRIÂNGULO

largura = 10

altura = 5

area = largura * altura

print ("A área do retângulo é:", area)

Nesse exemplo, declaramos as variáveis "largura" e "altura" e atribuímos a elas os valores 10 e 5, respectivamente. Em seguida, criamos uma terceira variável chamada "área" e atribuímos a ela o resultado da multiplicação entre "largura" e "altura". Por fim, utilizamos a função "print()" para exibir a mensagem "A área do retângulo é:" seguida do valor da variável "área".

VERIFICAÇÃO DE MAIORIDADE

idade = 18

if idade >= 18:

print ( "Você é maior de idade!")

else:

print("Você é menor de idade!")

Nesse exemplo, declaramos a variável "idade" e atribuímos a ela o valor 18. Em seguida, utilizamos uma estrutura condicional if-else para verificar se a idade é maior ou igual a 18. Se for, exibimos a mensagem "Você é maior de idade!". Caso contrário, exibimos a mensagem "Você é menor de idade!".

OPERADORES

Operadores em Python são símbolos ou palavras-chave que realizam operações sobre operandos (valores ou variáveis). Eles permitem realizar cálculos, comparações, operações lógicas e manipulações de dados. Aqui estão os principais tipos de operadores em Python:

OPERADORES ARITMÉTICOS

  • '+': Adição - Exemplo: '2 + 3' resulta em '5'

  • '-': Subtração - Exemplo: '5 - 2' resulta em '3'

  • '/': Divisão - Exemplo: '10 / 2' resulta em '5.0'

  • '*': Multiplicação - Exemplo: '3 * 4' resulta em '12'

  • '%': Módulo (resto da divisão) - Exemplo: '10 % 3' resulta em '1'

  • '**': Exponenciação (potência) - Exemplo: '2 ** 3' resulta em '8'

  • '//': Divisão inteira (quociente sem o resto) - Exemplo: '10 // 3' resulta em '3'

OPERADORES DE COMPARAÇÃO

  • '==': Igual a - Exemplo: '2 == 2' resulta em 'True'

  • '!=': Diferente de - Exemplo: '2 != 3' resulta em 'True'

  • '>': Maior que - Exemplo: '3 > 2' resulta em 'True'

  • '<': Menor que - Exemplo: '2 < 3' resulta em 'True'

  • '>=': Maior ou igual a - Exemplo: '3 >= 2' resulta em 'True'

  • '<=': Menor ou igual a - Exemplo: '2 <= 3' resulta em 'True'

OPERADORES LÓGICOS

  • 'and': "E" lógico (True se ambos os operandos forem True) - Exemplo: 'True and False' resulta em 'False'

  • 'or': "Ou" lógico (True se pelo menos um dos operandos for True) - Exemplo: 'True or False' resulta em 'True'

  • 'not': "Negação" lógica (inverte o valor lógico) - Exemplo: 'not True' resulta em 'False'

OPERADORES DE ATRIBUIÇÃO

  • '=': Atribuição simples - Exemplo: 'x = 5' atribui o valor '5' à variável 'x'

  • '+=': Adição e atribuição - Exemplo: 'x += 2' é equivalente a 'x = x + 2'

  • '-=': Subtração e atribuição - Exemplo: 'x -= 2' é equivalente a 'x = x - 2'

  • '*=': Multiplicação e atribuição - Exemplo: 'x *= 3' é equivalente a 'x = x * 3'

  • '/=': Divisão e atribuição - Exemplo: 'x /= 2' é equivalente a 'x = x / 2'

  • '%=': Módulo e atribuição - Exemplo: 'x %= 3' é equivalente a 'x = x % 3'

OPERADORES DE IDENTIDADE

  • 'is': verifica se dois objetos são o mesmo (mesma identidade) - Exemplo: 'x is y'

  • 'is not': verifica se dois objetos não são o mesmo - Exemplo: 'x is not y'

OPERADORES DE PERTENCIMENTO

  • 'in': verifica se um valor está presente em uma sequência (como uma lista ou string) - Exemplo: "'a' in 'apple'" resulta em `True`

  • 'not in': verifica se um valor não está presente em uma sequência - Exemplo: "'b' not in 'apple'" resulta em `True`

OPERADORES BIT A BIT (bitwise)

  • '&': 'E' bit a bit

  • '|': 'OU' bit a bit

  • '^': 'OU exclusivo' (XOR) bit a bit

  • '~': 'NEGAÇÃO' bit a bit

  • '<<': Deslocamento à esquerda

  • '>>': Deslocamento à direita

VANTAGENS DE PYTHON

Nesse momento, você deve estar se perguntando quais são as vantagens que essa linguagem traz. Como dito anteriormente, Python é uma linguagem muito utilizada, e nos oferece diversos benefícios. Veremos alguns deles a seguir:

A linguagem Python é utilizada em diversas áreas, incluindo web development, análise de dados, inteligência artificial, automação e mais. O Python também possui uma vasta gama de bibliotecas e frameworks (como Django, Flask, Pandas, NumPy) que aceleram o desenvolvimento. DESVANTAGENS DE PYTHON

Vimos que Python nos oferece diversos recursos vantajosos, a linguagem também tem seus defeitos e lados ruins. Iremos conhecer suas principais desvantagens a seguir.

Como o python é uma linguagem interpretada, ela acaba se tornando mais lenta em comparação com linguagens compiladas como o C ou C++. Isso pode ser um problema em aplicações que necessitam uma alta performance. Ele também pode acabar consumindo mais memória do que outras linguagens, ainda mais quando falamos de grandes aplicações. Isso ocorre devido à forma que essa linguagem gerencia objetos e memória internamente.

Por conta do GIL (Global Interpreter Lock), o suporte a multithreading em Python é limitado. Isso significa que em programas CPU-bound, o multithreading pode não resultar em ganhos de desempenho significativos. O python também não é a melhor escolha para desenvolvimento de aplicações móveis. Existem poucas bibliotecas e frameworks robustos para esse fim.

O gerenciamento de diferentes ambientes de Python pode ser complicado, especialmente quando se trabalha em projetos com múltiplas dependências. Por ser uma linguagem de tipagem dinâmica, muitos erros que poderiam ser detectados em tempo de compilação em outras linguagens só são descobertos em tempo de execução em Python, o que pode aumentar o tempo de depuração.

ESTRUTURAS DE CONTROLE

Estruturas de controle em Python são usadas para dirigir o fluxo de execução de um programa com base em condições específicas ou em repetições. As principais estruturas de controle em Python incluem condicionais e loops. Existem tipos de estruturas de controle, veremos a seguir.

CONDICIONAIS (if, elif, else)

As estruturas condicionais são usadas para executar diferentes blocos de código com base em condições específicas.

  • 'if': executa um bloco de código se a condição for verdadeira. O IF deve propor alguma coisa. É preciso escrever o IF e logo depois colocar a condição analisada. Então, em seguida, o bloco de comandos.

  • 'elif': executa um bloco de código se a condição anterior (`if` ou `elif`) for falsa e a nova condição for verdadeira. O elif é uma estrutura intermediária dentro da seção if-else no python e deve vir como um complemento a ambos. Quando você já tem um IF e um ELSE, mas precisa de uma condição para especificar outra regra, pode usar o elif.

  • 'else': executa um bloco de código se todas as condições anteriores (`if` e `elif`) forem falsas. O ELSE surge depois do IF, em complemento lógico a ele. Então, não existe hipótese de escrever um ELSE sem um IF antes. Geralmente, o ELSE não requer um teste, uma comparação, pois ele executa algo caso a comparação do IF não passe. Portanto, você só precisa declarar o ELSE e adicionar o bloco de comandos.

x = 10

if x > 0

print ("x é positivo")

elif x == 0:

print ("x é zero")

else:

print ("x é negativo")

Neste exemplo, a saída será `x é positivo` porque a condição `x > 0` é verdadeira.

LOOPS (for e while)

Os loops são usados para repetir um bloco de código várias vezes.

  • Loop 'for': o for é usado para iterar sobre uma sequência (como uma lista, tupla, string ou intervalo).

for i in range(5):

print(i)

Este código imprimirá os números de 0 a 4.

  • Loop 'while': o while continua a executar o bloco de código enquanto a condição for verdadeira.

count = 0

while count < 5:

print(count)

count += 1

Este código imprimirá os números de 0 a 4, incrementando 'count' até que a condição 'count < 5' se torne falsa.

COMANDOS DE CONTROLE DE LOOP (break, continue, else)

  • break: interrompe o loop antes de sua conclusão natural.

  • continue: pula o restante do bloco de código dentro do loop para a próxima iteração.

  • else:um bloco 'else' opcional pode ser adicionado após um loop 'for' ou 'while', que será executado se o loop for concluído normalmente (sem ser interrompido por um 'break').

for i in range(10):

if i == 5:

break # Para o loop quando i é 5

print(i)

O código acima imprimirá os números de 0 a 4 e depois sairá do loop devido ao 'break'.

for i in range(10):

if i % 2 == 0:

continue # Pula a iteração para números pares

print(i)

O código acima imprimirá apenas os números ímpares de 1 a 9.

for i in range(5):

print(i)

else :

print("Loop completo")

O código acima imprimirá os números de 0 a 4, seguido por "Loop completo".

COMPREENSÕES DE LISTA

Compreensões de lista são uma forma concisa de criar listas usando loops 'for' dentro de uma expressão.

squares = [x**2 for x in range(10)]

print(squares)

Este código criará uma lista dos quadrados dos números de 0 a 9.
